# Chowkbazarbd

## Project Overview

This is a B2B site where small businesses of our country buy products from china and vietnam in bulk

### Tech stacks
* Front-end
    * Angular
* Back-end
    * Laravel (API and admin panel)
* Admin panel
    * laravel voyager package
    * Bootstrap 4.1
